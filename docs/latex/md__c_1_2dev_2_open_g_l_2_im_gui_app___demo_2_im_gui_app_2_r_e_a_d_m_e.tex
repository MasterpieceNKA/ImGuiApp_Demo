 \href{https://github.com/MasterpieceNKA/ImGuiApp}{\texttt{ Im\+Gui\+App}} provides some basic functionality needed to quickly create C++ gui applications using the GLFW and Open\+GL backends of \href{https://github.com/ocornut/imgui}{\texttt{ Dear Im\+Gui}}. \href{https://github.com/MasterpieceNKA/ImGuiApp}{\texttt{ Im\+Gui\+App}} uses CMake and has been tested on Windows 11 (Min\+GW \& Clang 18.\+1.\+5) and Linux (Ubuntu 22.\+04 with GNU 11.\+4.\+0 compiler). \href{https://github.com/MasterpieceNKA/ImGuiApp_Demo}{\texttt{ Im\+Gui\+App Demo Application}} is a sample application developed using this project.

This project is inspired by \href{https://github.com/StudioCherno/Walnut}{\texttt{ Walnut}} which uses Vulkan instead of Open\+GL.\hypertarget{md__c_1_2dev_2_open_g_l_2_im_gui_app___demo_2_im_gui_app_2_r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{md__c_1_2dev_2_open_g_l_2_im_gui_app___demo_2_im_gui_app_2_r_e_a_d_m_e_autotoc_md1}
